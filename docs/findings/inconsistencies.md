## Format pro Eintrag

Wenn ein Eintrag hier "zugewiesen" wird, wird er mit folgender Struktur erweitert:

```markdown
### #X — [Kurzname]

**Doc A sagt:** ...
**Doc B sagt:** ...
**Warum ist das ein Problem?** ...
**Auflösung:** ...
**Wer hat es geklärt:** [Persona oder User]
**Status:** Offen / In Klärung / Aufgelöst
```
