# Goldoni – Operative Umsetzung

## Rahmenbedingungen

- Öffnungszeiten: Mittwoch bis Sonntag, 18:00–22:30 Uhr
- Ruhetage: Montag und Dienstag (Inhaber ggf. für Admin/Buchhaltung)
- Kein Mittagsbetrieb
- Produktion muss in bestehenden Betrieb passen — kein Zusatzpersonal

## Zeitfenster für Produktion

| Tag | Situation | Produktionsfenster |
|---|---|---|
| Montag | Ruhetag, Admin | Max. halber Tag |
| Dienstag | Ruhetag, Admin | Max. halber Tag |
| Mittwoch | Öffnung 18:00 | 09:00–15:00 — bestes Fenster |
| Donnerstag–Sonntag | Öffnung 18:00 | Theoretisch möglich, aber Erholung |

## Empfehlung: Option A — 1× pro Woche (Mittwoch)

Mittwoch 09:00–13:00 als Hauptproduktionsfenster:

1. Kein Abendstress davor
2. Frische Ware von Dienstag bereits eingelagert
3. MHD 7–8 Tage deckt Mittwoch bis Dienstag ab
4. Kein Produktionstag während Servicetagen

Kapazität: 40–50 Einheiten in 4 Stunden.

Option B (2× pro Woche: Dienstag + Mittwoch) erst wenn Nachfrage es rechtfertigt.

## Ablauf Produktionstag (Mittwoch)

```
09:00  Rohwaren aus Kühllager prüfen (Frische, Temperatur)
09:15  Produktion (Lasagne backen / Ragù kochen)
10:45  Abkühlung einleiten — flache GN-Behälter, max. 4 cm Schichthöhe → CCP 1
11:30  Vakuumieren (Produkt muss ≤ 10 °C sein) → CCP 2
11:45  Etikettieren (MHD eintragen, Charge protokollieren)
12:00  Einlagerung Kühlvitrine / Kühlschrank → CCP 3
12:15  Chargenprotokoll abschließen
12:30  Reinigung
```

Gesamtaufwand: ca. 3,5 Stunden inkl. Reinigung.

## Wochenrhythmus komplett

| Tag | Aktivität |
|---|---|
| Montag | Admin, Buchhaltung, Bestellung aufgeben (Metro + Di Gennaro) |
| Dienstag | Lieferung annehmen, Wareneingangskontrolle, Einlagerung |
| Mittwoch | 09:00–13:00 Produktion, 18:00 Service |
| Donnerstag–Sonntag | Abendservice, Verkauf, Tagesprotokoll Kühlung |

## Mengenplanung

| Produkt | Start (Woche) | Skalierung wenn >80 % verkauft |
|---|---|---|
| Lasagne al Forno | 30–40 Einheiten | +20 % pro Schritt, max. +30 % auf einmal |
| Ragù Bolognese | 15–20 Einheiten | analog |

Lieber ausverkauft als Vernichtung — Ausverkauft schafft Dringlichkeit.

## Personalaufwand

| Aufgabe | Wer | Zeit/Woche |
|---|---|---|
| Produktion | Küchenchef / Inhaber | 3–4 Std. |
| Vakuumieren + Etikettieren | Küchenhilfe oder Inhaber | 1 Std. |
| Tagesprotokoll | Verantwortliche Person | 15 Min./Tag |
| Verkauf (Erwähnen, Einpacken) | Servicepersonal | 0 Extraaufwand |

Gesamter Mehraufwand: ca. 5–6 Stunden/Woche.

## MHD-Restbestand-Regel

Produkte die 2 Tage vor MHD noch im Regal: als Tagesangebot ins Restaurant — kein Abfall, kein Verlust.

## Equipment-Checkliste

| Gerät | Status | Kosten wenn nötig |
|---|---|---|

| Etikettendrucker | Anschaffen | 200–400 € |
| Kernthermometer (digital) | Meist vorhanden | 20–40 € |
| Flache GN-Behälter 1/1 | Meist vorhanden | 15–30 € pro Stk |
| Kühlvitrine Verkauf | Optional Start | 800–1.500 € |
| Datumsstempel (Backup) | Anschaffen | 8 € |

## 8-Wochen-Rollout

| Woche | Meilenstein |
|---|---|
| 1–2 | Equipment bestellen, Veterinäramt kontaktieren, HACCP updaten |
| 3 | Testproduktion intern — Prozess testen, MHD validieren, selbst verkosten |
| 4 | Soft Launch: 20 Einheiten, nur Stammgäste, kein Marketing |
| 5–6 | Feedback auswerten, Menge auf 40 Einheiten erhöhen |
| 7–8 | Voller Launch: Instagram, WhatsApp, Google Business, Tischaufsteller |

Woche 3 ist entscheidend: Prozess einmal komplett durchlaufen, Produkt nach 7 Tagen selbst verkosten, Dokumentation prüfen — bevor der erste echte Verkauf stattfindet.
