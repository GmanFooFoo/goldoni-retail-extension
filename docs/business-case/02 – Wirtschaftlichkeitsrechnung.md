# Goldoni – Wirtschaftlichkeitsrechnung

## Prämisse

Hochwertige Zutaten (San Marzano D.O.P., Büffelmozzarella, Bio-Hackfleisch, Parmigiano Reggiano 24 Monate). Kein Discount-Segment.

## Investition (einmalig)

| Position | Kosten |
|---|---|
| Vakuumiergerät Profi (z.B. Henkelman) | 1.200–1.800 € |
| Etikettendrucker + LMIV-Software | 400–700 € |
| HACCP-Update durch Berater | 800–1.200 € |
| Registrierung Veterinäramt | 100–300 € |
| Erstausstattung Verpackungsmaterial | 300–500 € |
| Gesamt | ca. 3.000–4.500 € |

Amortisation bei 50 Einheiten/Woche: unter 3 Monate.
