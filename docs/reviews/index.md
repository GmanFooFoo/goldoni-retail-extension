# Reviews

Hier sammeln sich die **Prüfberichte** der 9 Personas, Dokument für Dokument.

Jeder Review-Bericht folgt einem festen Format:

- **Kurzurteil** (ein Satz)
- **Scoring** (1–5) über vier Dimensionen: Fachliche Korrektheit, Vollständigkeit, Umsetzbarkeit, Risiko-Abdeckung
- **Red Flags** — was die Persona sofort stört
- **Fundierte Kritikpunkte** — warum genau, mit Verweis auf Passagen
- **Was fehlt** — welche Informationen müssten noch da sein
- **Empfehlung** — Freigabe / Freigabe mit Auflagen / Rework / Stopp
